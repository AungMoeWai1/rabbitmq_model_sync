RabbitMQ Consumer Controller

Developed a model for managing RabbitMQ consumers directly from the Odoo backend. Each consumer can be linked to a specific Odoo model to determine where received data should be processed and stored. Supports start/stop actions and state tracking (draft, running, stopped).
